API

Usuarios

GET /admin/users Listar usuarios
GET /admin/users/:id Usuario por ID
PATCH /admin/users/:id Actualizar usuario
PATCH /admin/users/:id/status Cambiar estado
GET /admin/users/:id/pets Mascotas del usuario

Habitaciones

GET/admin/rooms Listar habitaciones
GET/admin/rooms/:id Habitación por ID
POST/admin/rooms Crear habitación
PATCH/admin/rooms/:id Actualizar habitación
PATCH/admin/rooms/:id/status Cambiar estado
DELETE/admin/rooms/:id Eliminar habitación

Servicios

GET/admin/services Listar servicios
GET/admin/services/:id Servicio por ID
POST/admin/services Crear servicio
PATCH/admin/services/:id Actualizar servicio
DELETE/admin/services/:id Eliminar servicio

Reservaciones

GET/admin/reservations Listar reservaciones
GET/admin/reservations/:id Reservación por ID
PATCH/admin/reservations/:id/status Cambiar estado
PATCH/admin/reservations/:id/check-in/confirm Confirmar check-in
PATCH/admin/reservations/:id/check-out/confirm Confirmar check-out

Pagos

GET/admin/payments Listar pagos
